Midi-Synth

MIDi-SYNTH


What is MidI-SYNTH:

Proof of concept for a browser-based MiDI synthesizer for manipulating sound and playing music.

What web APIs does it use?

  • - Web Audio API
  • - Web MidI API

What were the biggest challenges?

Figuring out how to translate concepts related to physical modular synthesis to code via the Web Audio API. Additionally, it took several attempts to produce a realistic looking virtual piano.

  2. Add capabilities to record sounds / songs.

  • - React.js
  • - Deployed to Surge

Objects In Space

Objects In Space (Team Project)

What is OIS?


Team project that sends daily push notifications regarding Near Earth Objects (NEOs). Statistics include relative size, velocity and whether or not the object is a potential threat.

What problem does it solve?

The potential threat from various objects in space is low but, nevertheless, present. Objects In Space gives people access to daily statistics related to NEOs, and allows them to understand the size and risks associated with them.

Who has this problem?

Anyone concerned about potential existential threats to humanity, or people with general curiosity about space.

What was the biggest challenges?

The biggest challenged faced by the team was figuring out how to send push notifications via Chrome Service Workers. Initially assigned to 1 of 3 team members, it was discovered to be more challenging than expected and took the whole team several hours over two days to find and implement a solution.

Technologies:

  • - Chrome Service Workers
  • - Angular.js
  • - Ruby On Rails
  • - Postgres
  • - Bootstrap
  • - Deployed to Heroku

youRecipe

youRecipe


What is YouRecipe?

YouRecipe is a convenient and easy to use app for creating, searching for, and storing recipes across different categories.

What problem does it solve?

YouRecipe replaces the old physical recipe box with a intuitive digital recipe catalog.

Who has this problem?

Anyone tired of maintaining a physical recipe box, and wanting an intuitive digital alternative.

What web APIs does it use?

  • - Food2Fork recipe database

What was the biggest challenge?

Creating a user experience that was intuitive and markedly better than maintaining a physical recipe box.

To Dos:

  1. Add image uploading for personal recipes.
  2. Create functionality to share recipes with friends.
  3. Add notes / comments section to recipe pages.

Technologies:

  • - Express.js
  • - Node.js
  • - MongoDB
  • - Materialize
  • - Deployed to Heroku
