Projects

Midi-Synth

MIDi-Synth

Description:

Browser-based MIDI synthesizer that leverages the Web Midi API to allow users to plug in a 37-key USB MIDI keyboard; users without access to a MIDI piano can use their mouse via the virtual piano keyboard.

Technologies:

  • - HTML5 Canvas
  • - CSS3
  • - Web Audio API
  • - Web Midi API
  • - React.js
  • - Deployed to Surge
Live Version | Github

Objects In Space

Objects In Space (Team Project)

Description:

Team project that tracks Near Earth Objects (NEOs). A user is sent a daily push notification with a list of NEOs; they can then click the notification to see the list and then click each NEO to see its stats (including relative size, velocity, and whether it is a potential threat or not).

Technologies:

  • - Chrome Service Workers
  • - Angular.js
  • - Ruby On Rails
  • - Postgres
  • - Bootstrap
  • - Deployed to Heroku
Live Version | Github
